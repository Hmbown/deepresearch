# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Security Advisories for this repository:

- Go to `Security` tab
- Click `Report a vulnerability`

If that is not available, open an issue with no exploit details and ask for a secure contact path.

## Scope

Please report issues related to:

- credential handling
- prompt/tool injection paths
- SSRF/network access controls
- dependency vulnerabilities with practical impact
